About Smart Child Monitor

The website has to provide information of the real-time location of a child, it's parent, it's route etcetera. It will push notifications when the child will distance a certain distance from a given point. Aditionally, notifications will be pushed regarding possible accidents such as collisions with ojects or autovehicles, when the child falls on the groud, or when it interracts with an animal.

Other functionalities

  • To access this site's functionalities you first have to register or login if you already have an account.

  • In the Monitor Page you can see information about the child.

  • In the Configuration Page you can set up the settings for the monitor page.
